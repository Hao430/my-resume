## 🔴 阻塞问题 (必须修复)

### 1. 🔴 CSS 变量命名系统不一致

**文件**：`src/components/*.vue`（旧组件）

**问题**：存在两套 CSS 变量命名系统：

| 新系统（使用中） | 旧系统（废弃） |
|-----------------|---------------|
| `--color-ink` | `--bg-primary` |
| `--color-text` | `--bg-secondary` |
| `--space-*` | `--spacing-*` |
| `--shadow-*` | `--shadow-*` |
| `--color-vermilion` | `--primary-color` |

**受影响的组件**：
- `Honors.vue` — 使用 `--bg-secondary`, `--bg-primary`, `--primary-color`, `--primary-light`, `--shadow-*`, `--border-*`
- `PersonalInfo.vue` — 使用 `--bg-secondary`, `--bg-tertiary`, `--primary-color`, `--spacing-*`
- `ProfessionalSummary.vue` — 使用 `--bg-primary`, `--bg-secondary`, `--text-primary`, `--text-secondary`
- `ProjectExperience.vue` — 使用 `--primary-color`, `--primary-light`, `--shadow-*`
- `Skills.vue` — 使用 `--bg-primary`, `--bg-secondary`, `--primary-color`, `--primary-light`, `--spacing-*`
- `WorkExperience.vue` — 使用 `--bg-primary`, `--bg-secondary`, `--text-primary`, `--text-secondary`, `--border-*`

**风险**：
- 这些组件的样式可能无法正确显示
- 全局样式覆盖可能不生效
- 维护困难

**建议**：
```css
/* 错误写法 */
background-color: var(--bg-secondary);

/* 正确写法 */
background-color: var(--color-ink-light);
```

### 2. 🔴 Hero.vue — 打字机效果使用非响应式变量

**文件**：`src/components/Hero.vue` 第 8-18 行

**问题**：
```typescript
let charIndex = 0  // ❌ 普通变量，不是响应式的

onMounted(() => {
  const typeInterval = setInterval(() => {
    if (charIndex < fullText.length) {
      displayText.value += fullText[charIndex]  // ✅ 响应式
      charIndex++  // ❌ 这个变化不会触发重新渲染
    }
  }, 150)
})
```

**风险**：打字机效果可能不稳定，或在某些情况下不工作

**建议**：
```typescript
const charIndex = ref(0)

onMounted(() => {
  const typeInterval = setInterval(() => {
    if (charIndex.value < fullText.value.length) {
      displayText.value += fullText.value[charIndex.value]
      charIndex.value++
    } else {
      clearInterval(typeInterval)
    }
  }, 150)
})
```

### 8. 🟡 AboutPage.vue — URL 判断逻辑不完整

**文件**：`src/pages/AboutPage.vue` 第 170-180 行

**问题**：
```typescript
{{ project.url.startsWith('http') ? '访问项目' : '查看详情' }}
```

**风险**：`startsWith('http')` 不够精确，`'http://example.com'` 和 `'https://example.com'` 都会匹配，但 `'http'` 本身也会匹配（虽然数据不太可能有这个问题）

**建议**：使用更精确的判断
```typescript
{{ project.url.startsWith('http://') || project.url.startsWith('https://') ? '访问项目' : '查看详情' }}
```
