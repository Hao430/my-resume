## 数据流（目标）

```
resume-data.json / 管理后台
        ↓ (种子脚本 / 写接口)
  ┌──────────┐
  │ PostgreSQL│ ←── Express API ←── Vue SPA
  └──────────┘
```

不再需要硬编码数据文件，内容更新不再触发构建部署。

## 部署方案

### 选项 A：前后端分离（推荐）

| 组件 | 部署目标 | 方式 |
|------|---------|------|
| 前端 `client/dist/` | 阿里云 ESA / CDN | Vite build → 静态文件 |
| 后端 `server/` | Railway / Fly.io | Express 容器化部署 |
| PostgreSQL | Supabase / Neon | 云数据库服务 |

### 选项 B：单机部署

```
Nginx → /api/* → localhost:3000 (Express)
      → /*     → /path/to/client/dist (静态文件)
```

## 安全事项

- API 写接口（POST/PUT/DELETE）当前无鉴权，仅适合个人站或内网使用
- 如需暴露到公网，建议前置 Nginx 做 IP 白名单或添加简单 Token 验证
